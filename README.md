BLACK-MANGO-BOT - شرح البوت

📌 تعريف:
BLACK-MANGO-BOT هو بوت Discord لإدارة التفاعل باستخدام نظام نقاط XP.
يعطي المستخدمين نقاط عند النشاط داخل السيرفر ويرفع مستواهم (Level).
كما يوفر نظام ترتيب (Leaderboard) لمتابعة أفضل الأعضاء.

----------------------------------------

⚙️ فكرة عمل البوت:
- منح XP عند إرسال الرسائل
- تخزين البيانات في MongoDB
- ترقية المستخدمين إلى مستويات أعلى
- عرض ترتيب الأعضاء

----------------------------------------

🚀 المميزات:
- نظام XP تلقائي
- نظام Levels (ترقية تلقائية)
- عرض ترتيب المستخدمين (Leaderboard)
- إنشاء روابط MongoDB ديناميكية
- سهل التعديل والتطوير

----------------------------------------

📂 الملفات الأساسية:
- example.js → الملف الرئيسي لتشغيل البوت
- package.json → المكتبات والإعدادات
- .env → يحتوي معلومات حساسة (لا يتم رفعه على GitHub)

----------------------------------------

🔐 ملف .env (سري):
BOT_TOKEN=YOUR_BOT_TOKEN
CLIENT_ID=YOUR_CLIENT_ID
MONGO_URI=YOUR_MONGO_URI
DB_NAME=discord-leveling
ATLAS_PUBLIC_KEY=YOUR_ATLAS_PUBLIC_KEY
ATLAS_PRIVATE_KEY=YOUR_ATLAS_PRIVATE_KEY
ATLAS_PROJECT_ID=YOUR_PROJECT_ID
CHANNEL_ID=YOUR_CHANNEL_ID

⚠️ ملاحظة: هذا الملف يحتوي بيانات حساسة، لا تشاركه أو ترفعه على GitHub.

----------------------------------------

🚀 طريقة التشغيل:
1. تثبيت المكتبات:
   npm install
2. إنشاء ملف .env ووضع البيانات الصحيحة
3. تشغيل البوت:
   node example.js

----------------------------------------

📜 أوامر البوت:
- Create Link → إنشاء رابط MongoDB جديد
- Delete Link → حذف الرابط الحالي
- My Links → عرض الرابط الخاص بك

----------------------------------------

⚠️ ملاحظات مهمة:
- تأكد من تغيير التوكن إذا تم تسريبه
- أضف ملف .env إلى .gitignore
- تأكد من تشغيل MongoDB بشكل صحيح
